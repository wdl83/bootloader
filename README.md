## HOWTO build

1. git clone https://github.com/wdl83/bootloader
1. cd bootloader
1. ./fetch_deps.sh
1. make

## HOWTO build (docker image - usefull for CI)
1. git clone https://github.com/wdl83/bootloader
1. cd bootloader
1. sudo docker build .
1. sudo docker run IMAGE_ID
